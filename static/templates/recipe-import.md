# MobileNet V2 Example

Example recipe using webnn/mobilenet-v2. Each row is one model file — change hf_model_id and file_path to your own models.

- [Model on Hugging Face](https://huggingface.co/webnn/mobilenet-v2)

| hf_model_id | file_path |
|---|---|
| webnn/mobilenet-v2 | onnx/mobilenet_v2.onnx |
| webnn/mobilenet-v2 | onnx/mobilenet_v2_fp16.onnx |
| webnn/mobilenet-v2 | onnx/mobilenet_v2_int8.onnx |
| webnn/mobilenet-v2 | onnx/mobilenet_v2_int4.onnx |
| webnn/mobilenet-v2 | tflite/mobilenet_v2.tflite |
| webnn/mobilenet-v2 | tflite/mobilenet_v2_fp16.tflite |
| webnn/mobilenet-v2 | litertlm/mobilenet_v2.litertlm |
| webnn/mobilenet-v2 | litertlm/mobilenet_v2_fp16.litertlm |
